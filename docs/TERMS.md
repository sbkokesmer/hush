## What Hush is

Hush is a privacy-focused web viewer that opens any website you add in an isolated session, blocks common trackers and ads, masks browser fingerprinting signals, and clears all browsing data when you close a tab.

## What Hush is not

- Hush is **not** a VPN. Hush does not hide your IP address from the websites you visit.
- Hush is **not** an antivirus. While it blocks file downloads by default and warns about known phishing sites, it cannot guarantee protection against every threat.
- Hush does **not** make you anonymous on the internet. If you sign in to a website through Hush, that website knows who you are.
