# Hush

A safer way to visit websites you don't fully trust.

Every site you add to Hush opens in a fully isolated session. Trackers and ads are blocked at the network layer, browser fingerprinting is masked, and the moment you close a tab every cookie, every cache entry, every trace is wiped clean — as if you were never there.

Hush has no servers. No analytics. No third-party SDKs that watch you. Everything stays on your device.
